# 管理端原型设计

本文件夹包含管理端各页面的HTML原型设计，用于开发前的UI参考。

## 文件列表

| 文件名                | 说明                 |
| --------------------- | -------------------- |
| `login.html`          | 登录页原型           |
| `dashboard.html`      | 工作台/仪表盘原型    |
| `parcel-inbound.html` | 包裹入库原型         |
| `parcel-list.html`    | 包裹列表原型         |
| `pickup.html`         | 出库核销原型         |
| `shelf.html`          | 货架管理原型         |
| `proxy.html`          | 代取订单原型         |
| `user.html`           | 用户管理原型         |
| `runner-audit.html`   | 跑腿员审核原型       |
| `stats.html`          | 数据统计原型         |
| `station.html`        | 驿站管理原型         |

## 使用说明

1. 直接在浏览器中打开HTML文件即可查看原型
2. 原型基于Element Plus样式，与实际页面效果一致
3. 原型中的数据为静态示例，仅供参考UI布局和交互

## 设计规范

- 统一使用Element Plus组件库
- 主题色：#409EFF（Element Plus默认蓝色）
- 圆角：4px
- 间距：基础单位8px
